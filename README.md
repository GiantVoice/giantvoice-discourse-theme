# GiantVoice Discourse Theme

Git-backed Discourse theme for `community.giantvoice.org`.

## Install

In Discourse admin:

1. Go to `Admin -> Customize -> Themes`.
2. Install from Git repository.
3. Use `https://github.com/GiantVoice/giantvoice-discourse-theme`.
4. Set the `GiantVoice Calm` color scheme.
5. Make `GiantVoice Community` the default theme.

## Design

The theme matches the GiantVoice app shell: calm blue-gray surfaces, subtle raised cards, GiantVoice blue as an accent, and a small cross-site navigation strip.
